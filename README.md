# 🔥 C++ 정렬 알고리즘 구현 및 테스트 시스템

3가지 정렬 알고리즘(삽입 정렬, 병합 정렬, 병합-삽입 정렬)을 C++로 구현하고, 자동 테스트 시스템을 통해 검증하는 프로젝트입니다.

## 📂 프로젝트 구조

```
sorting-algorithms/
├── sorting.cpp              # C++ 정렬 알고리즘 구현
├── makeCase.py        # 랜덤 테스트 데이터 생성기
├── validationCase.py            # 정렬 결과 검증기
├── input.txt               # 테스트 입력 데이터 (자동 생성됨)
├── output.txt              # 정렬 결과 출력 (C++ 실행 후 생성됨)
└── README.md               # 이 파일
```

## 🚀 빠른 시작

### 1단계: 테스트 데이터 생성

**중요! : 파이썬은 터미널에서 아래와 같은 명령줄을 입력해줘야 실행 가능합니다!**

```bash
# 기본: 10만개 랜덤 데이터 생성
python makeCase.py

# 사용자 지정: 5만개 정렬된 데이터 생성
python makeCase.py -s 50000 -p sorted
```

### 2단계: C++ 컴파일 및 실행
- 터미널을 이용해 컴파일을 해도 되고, 각자의 환경에 맞게 편하게 컴파일 및 실행을 하시면 됩니다.

```bash
# 컴파일
g++ -O2 -o sorting sorting.cpp

# 실행
./sorting
```

### 3단계: 결과 검증

```bash
# 전체 검증 및 상세 리포트
python validationCase.py

# 간단한 성공/실패만 확인
python validationCase.py --quiet
```

## 📊 랜덤 테스트 데이터 생성기 사용법

### 기본 사용법

```bash
python makeCase.py [옵션]
```

### 주요 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| `-s`, `--size` | 데이터 개수 | 100000 |
| `-p`, `--pattern` | 데이터 패턴 | random |
| `-o`, `--output` | 출력 파일명 | input.txt |
| `--seed` | 랜덤 시드 | 자동 |
| `--min` | 최솟값 | -1000000 |
| `--max` | 최댓값 | 1000000 |

### 데이터 패턴 종류

| 패턴 | 설명 | 용도 |
|------|------|------|
| `random` | 완전 랜덤 데이터 | 일반적인 성능 테스트 |
| `sorted` | 오름차순 정렬됨 | 삽입 정렬 Best Case |
| `reverse` | 내림차순 정렬됨 | 삽입 정렬 Worst Case |
| `partial` | 부분적으로 정렬됨 | 실제 데이터와 유사 |
| `duplicates` | 중복값 많음 | 중복 처리 테스트 |
| `nearly` | 거의 정렬됨 | 삽입 정렬 유리한 케이스 |

### 사용 예시

```bash
# 🔸 기본 테스트 (10만개 랜덤)
python makeCase.py

# 🔸 대용량 테스트 (50만개)
python makeCase.py -s 500000

# 🔸 Best Case 테스트 (정렬된 데이터)
python makeCase.py -s 10000 -p sorted

# 🔸 Worst Case 테스트 (역순 데이터)
python makeCase.py -s 10000 -p reverse

# 🔸 중복값 테스트
python makeCase.py -s 100000 -p duplicates

# 🔸 재현 가능한 테스트 (고정 시드)
python makeCase.py --seed 12345

# 🔸 값 범위 지정 테스트
python makeCase.py -s 10000 --min -1000 --max 1000

# 🔸 소규모 디버깅용
python makeCase.py -s 100 -o debug_input.txt
```

## ✅ 정렬 결과 검증기 사용법

### 기본 사용법

```bash
python validationCase.py [옵션]
```

### 주요 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| `-i`, `--input` | 입력 파일명 | input.txt |
| `-o`, `--output` | 출력 파일명 | output.txt |
| `-q`, `--quiet` | 간단한 출력 모드 | False |

### 검증 항목

#### 🔍 **출력 형식 검증**
- 정확히 3줄 구성 (각 알고리즘별 1줄)
- 각 줄 마지막 공백 없음
- 마지막 줄 끝 줄바꿈 없음
- 숫자 간 공백 구분자 정확성

#### 🎯 **정렬 정확성 검증**
- 입력/출력 데이터 크기 일치
- 모든 원소 보존 확인
- 오름차순 정렬 순서 확인
- 표준 정렬 결과와 일치 확인

#### 🔄 **알고리즘 일치성 검증**
- 3개 알고리즘 결과 동일성
- 불일치 시 구체적 차이점 분석

#### 📊 **데이터 특성 분석**
- 입력 데이터 통계 (크기, 범위, 중복값)
- 초기 정렬 상태 분석
- 무질서도(inversion count) 측정

### 사용 예시

```bash

python validationCase.py

# 🔸 사용자 지정 파일 검증
python validationCase.py -i test_data.txt -o result.txt

# 🔸 자동화용 간단 검증
python validationCase.py --quiet
```

## 🐛 문제 해결

### 자주 발생하는 문제

1. **"입력 파일을 찾을 수 없습니다"**
   ```bash
   # 해결: 테스트 데이터를 먼저 생성
   python makeCase.py
   ```

2. **"출력 파일을 찾을 수 없습니다"**
   ```bash
   # 해결: C++ 프로그램을 먼저 실행
   ./sorting
   ```

3. **"마지막 줄 끝에 불필요한 줄바꿈"**
   - C++ 코드에서 마지막 출력 후 `endl` 제거 필요

4. **"줄 끝에 불필요한 공백"**
   - C++ 코드에서 마지막 숫자 출력 후 공백 제거 필요
